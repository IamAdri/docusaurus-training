# Buttons

Here are some _built-in design_ options for <span style={{backgroundColor: "rgb(197, 125, 197)", color: "rgb(255, 255, 255)", fontWeight: "bold", padding: "0px 5px"}}>buttons</span> that you can import and use in your personal projects!
`toastType` is the prop that define if toast notification is either for success or error message. If you do not add this prop, `<Button></Button>` component will show a default toast type.

## Primary type of button

```tsx
import Button from "/src/components/DocsFeatures/Button";

<Button toastType="success" toastText="I have clicked on primary button!">
  Click on primary button
</Button>;
```

{<Button toastType="success" toastText="I have clicked on primary button!">
Click on primary button</Button>}

<br></br>
## Secondary type of button

```tsx
import Button from "/src/components/DocsFeatures/Button";

<Button
  design="secondary_button"
  toastText="I have clicked on secondary button!"
>
  Click on seconday button
</Button>;
```

{<Button design="secondary_button" toastText="I have clicked on secondary button!">
Click on seconday button</Button>}
